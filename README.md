# GameOfLife
Conway's game of Life

## USAGE: 
./life [options] ?<files>  
   - -custom   __*input a custom starting grid*  
   - -size     __*set size of grid (size > 1)*  
   - -iter     __*number of iterations (i > 0) default 20*  
   - -r        __*run the simulation*  
   - -help     __*Display this list of options*
